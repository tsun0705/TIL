## Character 형식
~~~
'nEWlec'
'A'
'123'
~~~
* CHAR(n) : 고정된 문자열, size : 1byte
* VARCHAR2(n) : 가변 길이 문자열, 최대 n만큼 크기를 갖을 수 있다, size : 1byte
* NCHAR(n) : 전 세계 문자 표현, size : 2~3 byte
* NVARCHAR2(n) : 전 세계 문자 표현, size : 2~3 byte

## Numeric 형식

* 그 외 다른 세계 1문자 = 3byte
