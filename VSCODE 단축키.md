## 에디터 창 제어
* 현재 창 닫기 Ctrl + w   
* 닫은 창 다시 열기 Ctrl + Shift + t   
* 사이드바 토글 Ctrl + b   
* 사이드바 탐색기 Ctrl + Shift + e   
* 사이드바 전체 검색 Ctrl + Shift + f   

## 소스코드 편집
* 들여쓰기 Tab, Ctrl + ]   
* 내어쓰기 Shift + Tab, Ctrl + [   
* 아래 행 삽입 Ctrl + Enter   
* 위에 행 삽입 Ctrl + Shift + Enter   
* 현재 행 이동 Alt + 화살표   
* 현재 행 복사 Alt + Shift + 화살표   
* 현재 행 삭제 Ctrl + Shift + k   
* 주석 Ctrl + /
