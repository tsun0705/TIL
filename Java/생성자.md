## 생성자
* 초기화를 위한 특별한 함수
* 객체가 생성 되자 마자 무조건 제일 먼저 실행되어야 한다.
* 생성될 때 단 한번만 실행되어야 한다.
* 생성자를 '하나'도 정의하지 않았으면 기본값인 null, 0 등으로 초기화된다. 
~~~
new ExamList();
~~~
