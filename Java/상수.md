## 상수(constant)
* 상수(constant)란 변수와 마찬가지로 데이터를 저장할 수 있는 메모리 공간을 의미
* 프로그램이 실행되는 동안 메모리에 저장된 데이터를 변경할 수 없음
* 상수는 선언과 동시에 반드시 초기화
* final 키워드를 사용
* 상수의 이름은 일반적으로 모두 대문자를 사용하여 선언
* 여러 단어로 이루어진 이름의 경우에는 언더스코어(_)를 사용

~~~
final int AGES = 30;
~~~

## 리터럴(literal)
* 그 자체로 값을 의미
~~~
int var = 30;         // 30이 바로 리터럴임.
final int AGES = 100; // 100이 바로 리터럴임.
~~~
