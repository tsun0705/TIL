## 컴퓨터

## 언어 번역
* 인터프리터 : 실시간, 대화형 언어, 스크립트 > 소스 공개 - JavaScript, VisualBasic, Python
* 컴파일러 : 일괄 처리, 오브젝트 생성 > 소스 모름 - C, C++, C#, JAVA, Fortran, COBOL
* 어셈블러 : 기호언어, 기계와 1대1 > unassembly - Assembly
* 소스 > 컴파일러 > 오브젝트 > 링크 (+라이브러리) > exe
