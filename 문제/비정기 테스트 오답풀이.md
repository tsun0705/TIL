## HTML/CSS 비정기테스트 210910

### 6번 accent-color
* 폼 요소에 색상을 적용할 수 있는 속성이다.

~~~
<style>
input {
  accent-color: auto;
  display: block;
  width: 30px;
  height: 30px;
}

input.custom {
  accent-color: rebeccapurple;
}
</style>

<input type="checkbox" checked />
<input type="checkbox" class="custom" checked />
~~~
