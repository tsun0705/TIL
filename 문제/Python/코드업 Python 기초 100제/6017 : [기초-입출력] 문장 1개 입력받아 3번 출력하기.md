### 문제 
```sh
정수(integer), 실수, 문자(character), 문자열(string) 등 1개만 입력받아 한 줄로 3번 출력해보자.
```
***

### 입력
```sh
1개의 데이터가 입력된다.
```
***

### 출력 
```sh
공백을 두고 3번 출력한다.
```
***

### 입력 예시
```sh
computer science
```
***

### 출력 예시
```sh
computer science computer science computer science
```
***

### 내 풀이
~~~
a = input()
print(a, a, a)
~~~
***

### 모범 답안
~~~
s=input()
print(s, s, s)
~~~
