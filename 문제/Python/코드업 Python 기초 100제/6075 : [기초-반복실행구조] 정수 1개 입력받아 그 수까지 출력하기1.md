### 문제 
```sh
정수(0 ~ 100) 1개를 입력받아 0부터 그 수까지 순서대로 출력해보자.
```

![image](https://user-images.githubusercontent.com/58898466/149271791-d83d0ce8-b97a-489a-8f25-e88405da715b.png)


### 내 풀이
~~~
a = int(input())
i = 0

while(i<=a) :
    print(i)
    i += 1
~~~

***
### 모범 답안
~~~
n=int(input())

i=0
while i<=n:
    print(i)
    i+=1

~~~
