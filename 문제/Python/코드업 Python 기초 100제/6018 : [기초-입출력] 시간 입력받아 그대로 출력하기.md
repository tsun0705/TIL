### 문제 
```sh
24시간 시:분 형식으로 시간이 입력될 때, 그대로 출력하는 연습을 해보자.
```
***

### 입력
```sh
시(hour) 분(minute)이 콜론(':')으로 구분되어 한 줄로 입력된다.
```
***

### 출력 
```sh
입력받은 시간 형식과 똑같이 "시:분" 형태로 출력한다.

```
***

### 입력 예시
```sh
3:16
```
***

### 출력 예시
```sh
3:16
```
***

### 내 풀이
~~~
a = input()
print(a)
~~~
***

### 모범 답안
~~~
a, b = input().split(':') 
print(a, b, sep=':')
~~~
