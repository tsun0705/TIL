### 문제 
```sh
실수 2개를 입력받아
합을 출력하는 프로그램을 작성해보자.
```
***

### 입력
```sh
2개의 실수가 줄을 바꿔 입력된다.
```
***

### 출력 
```sh
두 실수의 합을 출력한다.
```
***

### 입력 예시
```sh
0.1
0.9
```
***

### 출력 예시
```sh
1.0
```
***

### 내 풀이
~~~
a = input()
b = input()
print(float(a) + float(b))
~~~
***

### 모범 답안
~~~
a=input()
b=input()
a=float(a)
b=float(b)
c=a+b
print(c)
~~~
