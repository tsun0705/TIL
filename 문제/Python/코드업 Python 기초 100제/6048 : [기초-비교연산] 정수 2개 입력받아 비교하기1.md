### 문제 
```sh
두 정수(a, b)를 입력받아
a가 b보다 작으면 True 를, a가 b보다 크거나 같으면 False 를 출력하는 프로그램을 작성해보자.
```

![image](https://user-images.githubusercontent.com/58898466/148885602-1388a84b-2225-47bd-9dae-7f2d7c8ab1df.png)


### 내 풀이
~~~
a, b = input().split()
a = int(a)
b = int(b)
print(a < b)
~~~
