### 문제 
```sh
영문자 1개를 입력받아 10진수 유니코드(Unicode) 값으로 출력해보자.
```
***

### 입력
```sh
유니코드 영문자 1개가 입력된다.
```
***

### 출력 
```sh
유니코드 값을 10진수로 출력한다.
```
***

### 입력 예시
```sh
A
```
***

### 출력 예시
```sh
65
```
***

### 내 풀이
~~~
a = input()
a = ord(a)
print(a)
~~~
***

### 모범 답안
~~~
n=input()
n=ord(n)
print(n)
~~~
