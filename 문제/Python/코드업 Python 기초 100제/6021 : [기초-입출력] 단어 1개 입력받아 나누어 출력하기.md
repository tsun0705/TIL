### 문제 
```sh
알파벳과 숫자로 이루어진 단어 1개가 입력된다.
입력받은 단어의 각 문자를 한 줄에 한 문자씩 분리해 출력한다.
```
***

### 입력
```sh
5개의 문자로 이루어진 단어 1개가 입력된다.
```
***

### 출력 
```sh
각 문자를 한 줄에 한 문자씩 줄을 바꿔 출력한다.
```
***

### 입력 예시
```sh
Hello
```
***

### 출력 예시
```sh
H
e
l
l
o
```
***

### 내 풀이
~~~
a = input()
print(a[0])
print(a[1])
print(a[2])
print(a[3])
print(a[4])
~~~
***
