### 문제 
```sh
알파벳 문자와 숫자로 이루어진 단어 2개를 입력받아
순서대로 붙여 출력하는 프로그램을 작성해보자.
```
***

### 입력
```sh
알파벳과 숫자로 이루어진 2개의 단어가 공백으로 구분되어 입력된다.
```
***

### 출력 
```sh
입력된 2개의 단어를 순서대로 붙여 출력한다.
```
***

### 입력 예시
```sh
hello world
```
***

### 출력 예시
```sh
helloworld
```
***

### 내 풀이
~~~
a, b = input().split()
print(a + b)
~~~
***

### 모범 답안
~~~
a, b = input().split()
print(a+b)
~~~
