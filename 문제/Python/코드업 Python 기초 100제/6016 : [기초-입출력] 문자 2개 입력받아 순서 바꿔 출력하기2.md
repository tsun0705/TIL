### 문제 
```sh
공백을 두고 문자(character) 2개를 입력받아 순서를 바꿔 출력해보자.
```
***

### 입력
```sh
2개의 문자가 공백으로 구분되어 입력된다.
```
***

### 출력 
```sh
2개의 문자를 순서를 바꿔 한 줄로 출력한다.
```
***

### 입력 예시
```sh
a b
```
***

### 출력 예시
```sh
b a
```
***

### 내 풀이
~~~
a, b = input().split()
print(b, a)
~~~
***

### 모범 답안
~~~
a,b=input().split()
print(b)
print(a)
~~~
