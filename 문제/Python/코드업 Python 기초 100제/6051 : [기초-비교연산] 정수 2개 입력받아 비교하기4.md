### 문제 
```sh
두 정수(a, b)를 입력받아
a의 값이 b의 값과 서로 다르면 True 를, 같으면 False 를 출력하는 프로그램을 작성해보자.
```

![image](https://user-images.githubusercontent.com/58898466/148887350-a952971d-1579-4ede-80fd-799abd4708a6.png)


### 내 풀이
~~~
a, b = input().split()
a = int(a)
b = int(b)
print(a!=b)
~~~

***
### 모범 답안
~~~
a, b = input().split()
