### 문제 
```sh
실수 1개를 입력받아
소숫점 이하 두 번째 자리까지의 정확도로 반올림한 값을 출력해보자.
```
***
### 입력
```sh
실수 1개가 입력된다.
```
***
### 출력 
```sh
소숫점 이하 두 번째 자리까지의 정확도로 반올림한 값을 출력한다. 
```
***
### 입력 예시
```sh
3.141592
```
***
### 출력 예시
```sh
3.14
```
***
### 내 풀이
~~~
a = float(input())
print(format(a, ".2f"))
~~~

***
### 모범 답안
~~~
f = float(input())
print(round(f,2))
~~~
