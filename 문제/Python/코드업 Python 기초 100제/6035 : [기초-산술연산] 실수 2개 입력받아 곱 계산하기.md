### 문제 
```sh
실수 2개(f1, f2)를 입력받아 곱을 출력하는 프로그램을 작성해보자.
```
***
### 입력
```sh
2개의 실수가 공백으로 구분되어 입력된다.
```
***
### 출력 
```sh
첫 번째 실수와 두 번째 실수를 곱한 값을 출력한다.
```
***
### 입력 예시
```sh
0.5 2.0
```
***
### 출력 예시
```sh
1.0
```
***
### 내 풀이
~~~
a, b = input().split()
print(float(a) * float(b))
~~~

***
### 모범 답안
~~~
f1, f2 = input().split()
m = float(f1)*float(f2)
print(m)
~~~
