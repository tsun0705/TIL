### 문제 
```sh
실수(real number) 1개를 입력받아 줄을 바꿔 3번 출력해보자.
```
***

### 입력
```sh
실수 1개가 입력된다.
```
***

### 출력 
```sh
입력받은 실수를 줄을 바꿔 3번 출력한다.
```
***

### 입력 예시
```sh
0.1
```
***

### 출력 예시
```sh
0.1
0.1
0.1
```
***

### 내 풀이
~~~
a = float(input())
print(a)
print(a)
print(a)
~~~
***

### 모범 답안
~~~
a=input()
print(a)
print(a)
print(a)
~~~
