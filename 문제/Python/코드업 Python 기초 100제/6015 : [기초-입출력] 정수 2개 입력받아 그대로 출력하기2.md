### 문제 
```sh
공백을 두고 입력된정수(integer) 2개를 입력받아 줄을 바꿔 출력해보자.
```
***

### 입력
```sh
2개의 정수가 공백으로 구분되어 입력된다.

```
***

### 출력 
```sh
입력된 두 정수를 줄을 바꿔 출력한다.
```
***

### 입력 예시
```sh
1 2
```
***

### 출력 예시
```sh
1
2
```
***

### 내 풀이
~~~
a, b = input().split()
print(a)
print(b)
~~~
***

### 모범 답안
~~~
a,b=input().split()
print(a)
print(b)
~~~
