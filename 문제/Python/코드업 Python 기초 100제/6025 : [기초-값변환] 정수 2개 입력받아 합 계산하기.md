### 문제 
```sh
정수 2개를 입력받아
합을 출력하는 프로그램을 작성해보자.
```
***

### 입력
```sh
2개의 정수가 공백으로 구분되어 입력된다.
```
***

### 출력 
```sh
두 정수의 합을 출력한다.
```
***

### 입력 예시
```sh
123 -123
```
***

### 출력 예시
```sh
0
```
***

### 내 풀이
~~~
a, b = input().split()
print(int(a) + int(b))
~~~
***

### 모범 답안
~~~
a, b = input().split()
a=int(a)
b=int(b)
c=a+b
print(c)
~~~
