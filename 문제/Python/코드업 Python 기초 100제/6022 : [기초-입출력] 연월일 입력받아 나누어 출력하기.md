### 문제 
```sh
6자리의 연월일(YYMMDD)을 입력받아 나누어 출력해보자.
```
***

### 입력
```sh
6자리 숫자로 이루어진 연월일(YYMMDD)이 입력된다.
```
***

### 출력 
```sh
년도(YY) 월(MM) 일(DD)을 공백으로 구분해 한 줄로 출력한다.
```
***

### 입력 예시
```sh
200304
```
***

### 출력 예시
```sh
20 03 04
```
***

### 내 풀이
~~~
a = input()
print(a[0:2] + ' ' + a[2:4] + ' ' + a[4:6])  
~~~
***

### 모범 답안
~~~
s = input()
print(s[0:2], s[2:4], s[4:6], sep=' ')
~~~
