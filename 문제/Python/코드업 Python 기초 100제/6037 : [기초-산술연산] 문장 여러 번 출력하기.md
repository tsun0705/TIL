### 문제 
```sh
반복 횟수와 문장을 입력받아 여러 번 출력해보자.
```
***
### 입력
```sh
반복 횟수와 문장이 줄을 바꿔 입력된다.
```
***
### 출력 
```sh
입력된 횟수만큼 입력된 문장을 출력한다.
```
***
### 입력 예시
```sh
3
I love CS
```
***
### 출력 예시
```sh
I love CSI love CSI love CS
```
***
### 내 풀이
~~~
a = input()
b = input()
print(int(a) * b)
~~~

***
### 모범 답안
~~~
n = input()
s = input()
print(int(n)*s)
~~~
