### 문제 
```sh
실수 2개(f1, f2)를 입력받아
f1을 f2번 거듭제곱한 값을 출력하는 프로그램을 작성해보자.
```
***
### 입력
```sh
2개의 실수(f1, f2)가 공백으로 구분되어 입력된다.
```
***
### 출력 
```sh
f1을 f2번 거듭제곱한 값을 출력한다.
```
***
### 입력 예시
```sh
4.0 2.0
```
***
### 출력 예시
```sh
16.0
```
***
### 내 풀이
~~~
a, b = input().split()
c = float(a) ** float(b)
print(c)
~~~

***
### 모범 답안
~~~
f1, f2 = input().split()
f3 = float(f1)**float(f2)
print(f3)
~~~
