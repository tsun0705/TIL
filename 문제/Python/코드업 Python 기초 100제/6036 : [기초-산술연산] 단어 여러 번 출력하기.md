### 문제 
```sh
단어와 반복 횟수를 입력받아 여러 번 출력해보자.
```
***
### 입력
```sh
단어와 반복 횟수가 공백으로 구분되어 입력된다.
```
***
### 출력 
```sh
입력된 단어를 입력된 횟수만큼 반복해 출력한다.
```
***
### 입력 예시
```sh
love 3
```
***
### 출력 예시
```sh
lovelovelove
```
***
### 내 풀이
~~~
a, b = input().split()
print(a * int(b))
~~~

***
### 모범 답안
~~~
w, n = input().split()
n=int(n)
print(w*n)
~~~
