### 문제 
```sh
실수 2개(f1, f2)를 입력받아
f1 을 f2 로 나눈 값을 출력해보자. 이 때 소숫점 넷째자리에서 반올림하여 무조건 소숫점 셋째 자리까지 출력한다.
```
***
### 입력
```sh
2개의 실수(f1, f2)가 공백으로 구분되어 입력된다.
```
***
### 출력 
```sh
f1을 f2로 나눈 결과를 소숫점 이하 넷째 자리에서 반올림하여 소숫점 세 번째 자리까지 출력한다.
```
***
### 입력 예시
```sh
10.0 3.0
```
***
### 출력 예시
```sh
3.333
```
***
### 내 풀이
~~~
a,b=input().split()
a=float(a)
b=float(b)
print(format(a/b,".3f"))
~~~
 
***
### 모범 답안
~~~
a,b=input().split()
a=float(a)
b=float(b)
c=a/b
print('%.3f'%c)
~~~
