### 문제 
```sh
"연도.월.일"을 입력받아 "일-월-연도" 순서로 바꿔 출력해보자.
```
***

### 입력
```sh
연도, 월, 일이 닷('.')으로 구분되어 입력된다.
```
***

### 출력 
```sh
대시(마이너스 기호)를 구분기호로 사용해서
일-월-연도로 바꿔 출력한다.
```
***

### 입력 예시
```sh
2020.3.4
```
***

### 출력 예시
```sh
4-3-2020
```
***

### 내 풀이
~~~
y, m, d = input().split('.') 
print(d+"-"+m+"-"+y)
~~~
***

### 모범 답안
~~~
a, b, c = input().split('.')
print(c,b,a,sep='-')
~~~
