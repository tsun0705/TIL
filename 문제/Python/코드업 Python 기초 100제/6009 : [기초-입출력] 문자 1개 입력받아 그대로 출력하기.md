### 문제 
```sh
문자(character)는
0~9, a~z, A~Z, !, @, #, {, [, <, ... 과 같이 
길이가 1인 기호라고 할 수 있다.

변수에 문자 1개를 저장한 후
변수에 저장되어 있는 문자를 그대로 출력해보자.
```
***

### 입력
```sh
문자 1개가 입력된다.
```
***

### 출력 
```sh
입력된 문자를 그대로 출력한다.
```
***

### 입력 예시
```sh
a
```
***

### 출력 예시
```sh
a
```
***

### 내 풀이
~~~
a = input()
print(a)
~~~
***

### 모범 답안
~~~
c=input()
print(c)
~~~
