### 문제 
```sh
정수 2개(a, b)를 입력받아
a를 b번 곱한 거듭제곱을 출력하는 프로그램을 작성해보자.
```
***
### 입력
```sh
2개의 정수(a, b)가 공백으로 구분되어 입력된다.
```
***
### 출력 
```sh
a를 b번 거듭제곱한 값을 출력한다.
```
***
### 입력 예시
```sh
2 10
```
***
### 출력 예시
```sh
1024
```
***
### 내 풀이
~~~
a, b = input().split
print(int(a) ** int(b))
~~~

***
### 모범 답안
~~~
a, b = input().split()
c = int(a)**int(b)
print(c)
~~~
