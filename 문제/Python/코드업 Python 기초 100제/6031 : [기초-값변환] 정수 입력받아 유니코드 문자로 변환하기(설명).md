### 문제 
```sh
10진 정수 1개를 입력받아
유니코드 문자로 출력해보자.
```
***
### 입력
```sh
10진 정수 1개(32 ~ 126 범위)가 입력된다.
```
***
### 출력 
```sh
유니코드 문자로 출력한다.
```
***
### 입력 예시
```sh
65
```
***
### 출력 예시
```sh
A
```
***
### 내 풀이
~~~
a = input()
a = int(a)
print(chr(a))
~~~

***
### 모범 답안
~~~
c=input()
c=int(c)
print(chr(c))
~~~
