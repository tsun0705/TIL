## 튜플
* 여러 데이터를 동시에 저장이 가능
* 자료형이 일치하지 않아도 됨
* 다른 튜플까지 포함 가능
* 생성 후 내부의 데이터 삭제 및 수정 불가능
* 프로그램 실행 중 절대 변하지 않거나 변해서는 안되는 여러 값들을 저장
* 튜플끼리의 더하기 연산(+)은 두 튜플의 원소만을 순서대로 가지는 새로운 튜플을 생성하여 반환

~~~
a = (1,"hello")
a2 = (3,4.22)
a3 = t1 + t2
a4 = t1 * 3
a5 = t3[1]

print(a)
print(a2)
print(a3)
print(a4)
print(a5)
~~~
~~~
실행 결과: 
(1, 'hello')
(3, 4.22)
(1, 'hello', 3, 4.22)
(1, 'hello', 1, 'hello', 1, 'hello')
hello
~~~
